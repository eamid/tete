MATLAB code for t-Exponential Triplet Embedding (t-ETE) algorithm

Reference:
E. Amid, N. Vlassis, and M. Warmuth, "Low-dimensional Data Embedding via
Robust Ranking", https://arxiv.org/pdf/1611.09957.pdf

Contents:

data.mat      ---     synthetic dataset of points from two distinct clusters
tete.m        ---     applies the t-ETE algorithm
tete_grad.m   ---     calculates the gradient for t-ETE
genTriplet.m  ---     generates synthetic triplets
demo.m        ---     a working example for the t-ETE algorithm
